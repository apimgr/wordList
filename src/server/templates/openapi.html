API Documentation

WordList REST API v1 - EFF Large Wordlist for Passphrases

Base URL

{{.BaseURL}}/api/v1

Format

JSON by default, add .txt extension for plain text

Authentication

No authentication required - API is freely available

Passphrase Generation

GET /api/v1/passphrase

Generate a passphrase with 6 words (default)

Example Response

{
  "passphrase": "Correct-Horse-Battery-Staple-Widget-Gizmo",
  "words": ["Correct", "Horse", "Battery", "Staple", "Widget", "Gizmo"],
  "word_count": 6,
  "separator": "-",
  "capitalized": true
}
GET /api/v1/passphrase/{count}

Generate a passphrase with a specific number of words

Path Parameters

NameTypeDescription
countintegerNumber of words (3-20)

Query Parameters

NameTypeDefaultDescription
separatorstring-Word separator character
capitalizebooleantrueCapitalize first letter of each word

Examples

# 8-word passphrase with underscores
curl "{{.BaseURL}}/api/v1/passphrase/8?separator=_"

# 4-word passphrase, lowercase
curl "{{.BaseURL}}/api/v1/passphrase/4?capitalize=false"

# Plain text output
curl {{.BaseURL}}/api/v1/passphrase/6.txt

Random Words

GET /api/v1/random

Get 6 random words (default)

GET /api/v1/random/{count}

Get a specific number of random words

Path Parameters

NameTypeDescription
countintegerNumber of words (1-100)

Example

curl {{.BaseURL}}/api/v1/random/10

Search

GET /api/v1/search/{query}

Search for words by prefix or substring

Path Parameters

NameTypeDescription
querystringSearch term

Query Parameters

NameTypeDefaultDescription
modestringprefixSearch mode: "prefix" or "contains"
limitinteger100Maximum results (1-1000)

Examples

# Words starting with "pass"
curl {{.BaseURL}}/api/v1/search/pass

# Words containing "word"
curl "{{.BaseURL}}/api/v1/search/word?mode=contains"

Browse

GET /api/v1/words

Get all words (paginated)

Query Parameters

NameTypeDefaultDescription
pageinteger1Page number
per_pageinteger100Words per page (1-1000)
GET /api/v1/letter/{letter}

Get all words starting with a specific letter

Example

curl {{.BaseURL}}/api/v1/letter/a
GET /api/v1/length/{length}

Example

curl {{.BaseURL}}/api/v1/length/5

GET /api/v1/count
